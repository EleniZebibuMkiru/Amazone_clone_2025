# Amazon Clone Project

This is a **full-stack Amazon clone** built using **React**, **Vite**, and other modern web technologies. The project is a practice application to learn about e-commerce UI, routing, and component-based development.

## Overview

The project replicates the front-end UI of Amazon with components like:

- Header and navigation bar  
- Product carousel  
- Product listings (mock data)  
- Responsive design for mobile and desktop  

It helps to understand React components, routing with `react-router-dom`, and integrating image carousels.

---

## Features

- **Header Component:** Navigation, logo, search bar  
- **Carousel Component:** Auto-playing image slider  
- **Responsive Design:** Works on mobile and desktop  
- **Reusable Components:** For scalable front-end development  
- **Image Assets:** Local images for carousel and products  

---

## Technologies Used

- **React** – Front-end library  
- **Vite** – Build tool for fast development  
- **React Responsive Carousel** – Carousel library  
- **CSS Modules** – Component-level styling  
- **JavaScript ES6+**
